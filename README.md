# Интерфейс ИИС(первого по списку брокерского счета) Тбанка

### Системные требования

NodeJS 18.20.5 (через export NODE_OPTIONS=--openssl-legacy-provider)

### Инструкция для запуска

1. Находясь в корневом каталоге, выполнить установку зависимостей при помощи команды

```
npm install
```

2. Запустить локально проект при помощи команды

```
npm start
```

3. Перейти в браузере на локальный хост

```
http://localhost:3000
```
